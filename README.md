# Сайт-приглашение на свадьбу

Одностраничный свадебный сайт-приглашение Романа и Анастасии.

## О проекте

Проект представляет собой scroll-лендинг с несколькими тематическими блоками:

- первый экран с датой, именами и обратным отсчетом;
- блок приглашения с текстом и фото;
- блок «Жених / Невеста» и подтверждение присутствия;
- блок «Ждем Вас на свадьбе!» с палитрой цветов для дресс-кода.

Визуально сайт построен на светлом фоне с декоративным паттерном из букв `L O V E`.

## Технологии

- `Vite`
- `TypeScript`
- `Vanilla CSS`

## Запуск локально

```bash
npm install
npm run dev -- --host --port 4173
```

Открыть в браузере: `http://127.0.0.1:4173`.

## Сборка

```bash
npm run build
```
