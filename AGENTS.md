# AGENTS.md

## Назначение

Этот репозиторий содержит фронтенд свадебного одностраничного сайта-приглашения.

## Текущий стек

- `Vite`
- `TypeScript`
- `Vanilla CSS` (без UI-библиотек)
- статические ассеты в `public/images`

## Правила работы

1. Сохранять текущий визуальный стиль проекта (светлый минимализм + декоративный LOVE-паттерн).
2. Не добавлять тяжелые зависимости без явной необходимости.
3. При изменениях UI обязательно проверять результат через **Playwright MCP**:
   - использовать инструменты `mcp__playwright__*`;
   - открыть страницу локально;
   - проверить mobile и desktop viewport;
   - убедиться, что верстка не сломана и ключевые блоки отображаются корректно.
4. После правок запускать сборку:
   - `npm run build`
